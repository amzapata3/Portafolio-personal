PROYECTOS

El proyecto Internships4U SENA consistió en el desarrollo de una plataforma web de búsqueda de empleo para estudiantes universitarios. El desarrollo abarcó todo el ciclo de vida de un software, comenzando desde la fase de análisis y planificación, se diseñó la estructura y la interfaz de usuario de la plataforma, utilizando HTML y CSS para una maquetación limpia y responsiva. Para la parte interactiva y dinámica del sitio, se utilizó Vue 3. Durante la fase de desarrollo, se implementaron pruebas de testeo exhaustivas para asegurar la calidad y el funcionamiento adecuado de la plataforma. También se generaron diagramas de flujo, diagramas de casos de uso y documentación técnica.

Desarrolle una interfaz para un chatbot de Enseña por Colombia utilizando HTML, CSS y JavaScript. Esta interfaz está conectada al backend mediante una API Flask, lo que permite la interacción en tiempo real con el servidor. El chatbot fue diseñado para responder preguntas frecuentes de los usuarios, ofreciendo respuestas rápidas y precisas a las consultas más comunes.

Desarrolle un chatbot de WhatsApp utilizando la librería Baylies y el lenguaje de programación JavaScript. El chatbot fue diseñado con un menú principal que ofrece opciones con respuestas automáticas, las cuales derivan a submenús según la elección del usuario, cuenta con respuestas predefinidas para cuando el usuario selecciona una opción incorrecta, informando que la opción no es válida.Durante el desarrollo, se construyeron diagramas de flujo para visualizar y organizar las rutas de interacción entre el usuario y el sistema. El proyecto también incluyó pruebas manuales para verificar su funcionamiento y documentación.
